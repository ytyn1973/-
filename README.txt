一、项目简介
项目的目的是设计并制作一个能够识别并抓取地上物品的起重机，作为学校的一个课程设计。
项目是以全国大学生起重机创意竞赛为背景的。
项目的要求是起重机能够自动识别地上摆放的可乐或者保鲜盒，并将其抓取并移动到指定位置，同时保证安全和稳定。
项目的分工是由七个人组成的小组合作完成，分电控，软件。机械三个方向。
本人负责的部分是视觉模块的设计和实现，主要包括使用工业相机采集图像，使用opencv进行图像处理和轮廓识别，使用串口与上位机通信等。
二 环境配置
硬件环境：
相机：海康MV-CA016-10UMUV相机+镜头
开发板：JETSON tx2
串口：tx2上的串口（THS2）
起重机：stm32单片机+电机+机械部分
软件环境：
操作系统：Ubuntu 18.04
编程语言：Python 3.6
库：海康工业相机的sdk
