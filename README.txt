# 项目简介

本项目的目的是设计并制作一个能够识别并抓取地上物品的起重机，作为学校的一项课程设计。项目背景是全国大学生起重机创意竞赛。

## 项目要求
起重机应能自动识别地上摆放的可乐罐或保鲜盒，并将其抓取并移动到指定位置，同时确保安全和稳定。

## 项目分工
项目由7人小组合作完成，分为电控、软件、机械三个方向。

- **本人负责部分**: 视觉模块的设计与实现，主要工作包括：
  - 使用工业相机采集图像
  - 使用 OpenCV 进行图像处理与轮廓识别
  - 使用串口与上位机通信

# 环境配置

## 硬件环境
- **相机**: 海康 MV-CA016-10UMUV 相机 + 镜头
- **开发板**: JETSON TX2
- **串口**: TX2 上的串口（THS2）
- **起重机**: STM32 单片机 + 电机 + 机械部分

## 软件环境
- **操作系统**: Ubuntu 18.04
- **编程语言**: Python 3.6
- **库**: 海康工业相机 SDK
